# Memory Plugin Stub

This plugin will add optional external memory service integration for PKA.

## User Value

Users get durable retrieval context across sessions without making any
specific memory service mandatory in PKA core.

## Planned Capabilities

- Connect to a configured memory endpoint
- Retrieve relevant prior context for routing and validation
- Store selected evidence-backed learnings
- Support retrieval-augmented context workflows

## Status

Stub only. No code is installed by this plugin yet.
